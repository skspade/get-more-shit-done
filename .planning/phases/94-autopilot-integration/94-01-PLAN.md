# Plan 94-01: runAutomatedUAT() and auditAndUAT() in autopilot.mjs

## Objective

Add the `runAutomatedUAT()` function and `auditAndUAT()` helper to autopilot.mjs, wiring automated UAT into the milestone completion flow at all three insertion points. After this plan, the autopilot pipeline runs UAT after audit passes and routes results to completion or gap closure.

## Context

@get-shit-done/scripts/autopilot.mjs
@.planning/phases/94-autopilot-integration/94-CONTEXT.md
@.planning/research/ARCHITECTURE.md

## Must-Haves (goal-backward)

1. `runAutomatedUAT()` function exists in autopilot.mjs and follows the `runMilestoneAudit()` pattern
2. `auditAndUAT()` helper replaces all three audit-to-completion sites
3. UAT pass (0) proceeds to completion, UAT gaps (10) feeds gap closure, UAT crash uses debug retry
4. Non-web projects (no uat-config.yaml) skip UAT silently with return 0

<task id="1">
<title>Add runAutomatedUAT() function</title>
<actions>
Add `runAutomatedUAT()` async function to autopilot.mjs, placed after `runMilestoneAudit()` (after line ~922). The function:

1. Gates on uat-config.yaml existence:
   ```javascript
   const configPath = path.join(PROJECT_DIR, '.planning', 'uat-config.yaml');
   if (!fs.existsSync(configPath)) {
     logMsg('UAT: skipped (no uat-config.yaml)');
     console.log('No uat-config.yaml found. Skipping automated UAT.');
     return 0;
   }
   ```

2. Prints banner and invokes workflow:
   ```javascript
   printBanner('AUTOMATED UAT');
   const uatExit = await runStepWithRetry('/gsd:uat-auto', 'automated-uat');
   if (uatExit !== 0) {
     logMsg(`UAT: workflow failed exit=${uatExit}`);
     return 1;
   }
   ```

3. Reads MILESTONE-UAT.md frontmatter for status:
   ```javascript
   const uatFile = path.join(PROJECT_DIR, '.planning', 'MILESTONE-UAT.md');
   if (!fs.existsSync(uatFile)) {
     logMsg('UAT: no MILESTONE-UAT.md produced');
     return 1;
   }
   const uatStatus = (await gsdTools('frontmatter', 'get', uatFile, '--field', 'status', '--raw')).trim();
   logMsg(`UAT: result=${uatStatus}`);
   ```

4. Routes based on status:
   ```javascript
   switch (uatStatus) {
     case 'passed':
       printBanner('UAT PASSED');
       return 0;
     case 'gaps_found':
       printBanner('UAT: FAILURES FOUND');
       return 10;
     default:
       return 1;
   }
   ```

This follows the exact pattern of `runMilestoneAudit()` with the same exit code contract (0/10/1).

<task id="2">
<title>Add auditAndUAT() helper and wire insertion points</title>
<actions>
1. Add `auditAndUAT()` helper function after `runAutomatedUAT()`:
   ```javascript
   async function auditAndUAT() {
     const auditResult = await runMilestoneAudit();
     if (auditResult !== 0) return auditResult;
     const uatResult = await runAutomatedUAT();
     return uatResult;
   }
   ```

2. Replace insertion point 1 (lines ~1088-1099, "all phases complete on startup"):
   Change from:
   ```javascript
   const auditResult = await runMilestoneAudit();
   if (auditResult === 0) {
     await runMilestoneCompletion();
     process.exit(0);
   } else if (auditResult === 10) {
     await runGapClosureLoop();
     await runMilestoneCompletion();
     process.exit(0);
   } else {
     console.error('ERROR: Milestone audit encountered an error');
     process.exit(1);
   }
   ```
   To:
   ```javascript
   const result = await auditAndUAT();
   if (result === 0) {
     await runMilestoneCompletion();
     process.exit(0);
   } else if (result === 10) {
     await runGapClosureLoop();
     await runMilestoneCompletion();
     process.exit(0);
   } else {
     console.error('ERROR: Milestone audit or UAT encountered an error');
     process.exit(1);
   }
   ```

3. Replace insertion point 2 (lines ~1165-1176, "phases complete during main loop"):
   Same pattern -- replace `runMilestoneAudit()` with `auditAndUAT()`.

4. Replace insertion point 3 (line ~1004, "after gap closure re-audit"):
   Inside `runGapClosureLoop()`, replace:
   ```javascript
   const auditResult = await runMilestoneAudit();
   ```
   With:
   ```javascript
   const auditResult = await auditAndUAT();
   ```
   This ensures re-UAT happens automatically after gap closure fixes pass re-audit.
</actions>
</task>
