# Plan 45-01: Add cmdInitPrReview to gsd-tools init dispatch

## Goal

The pr-review workflow calls `gsd-tools.cjs init pr-review` but no such subcommand exists, causing a runtime crash that blocks the entire quick route. Add the missing `cmdInitPrReview` function (identical to `cmdInitLinear`), wire it into the init switch, and add tests.

## must_haves

- `gsd-tools.cjs init pr-review` returns valid JSON with all fields the pr-review workflow expects
- The init switch statement routes `pr-review` to `cmdInitPrReview`
- The error message's "Available:" list includes `pr-review`
- Tests cover all output fields matching the `cmdInitLinear` test pattern
- Quick route E2E path no longer crashes at the init step
