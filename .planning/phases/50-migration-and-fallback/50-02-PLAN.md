# Plan 50-02: Entrypoint Rewrite and Resume Instruction Updates

## Objective

Rewrite `bin/gsd-autopilot` to route to `autopilot.mjs` by default and to `autopilot-legacy.sh` with `--legacy`, and update all resume/escalation instruction references in `autopilot.mjs` from `autopilot.sh` to `gsd-autopilot`.

## must_haves

- `bin/gsd-autopilot` routes to `autopilot.mjs` by default using `exec node`
- `bin/gsd-autopilot` routes to `autopilot-legacy.sh` when first argument is `--legacy` (shifts the flag before exec)
- `bin/gsd-autopilot` validates the target script exists before exec, printing an error and exiting 1 if missing
- `bin/gsd-autopilot` remains a bash script with `#!/usr/bin/env bash` shebang
- `SCRIPTS_DIR` resolves to `$HOME/.claude/get-shit-done/scripts`
- All 6 `autopilot.sh` references in `autopilot.mjs` are replaced with `gsd-autopilot`
- No other files are modified

## Tasks

<task id="1">
<title>Rewrite bin/gsd-autopilot entrypoint</title>
<actions>
1. Read the current `bin/gsd-autopilot` file
2. Replace the entire file content with the new entrypoint script:
   - Keep `#!/usr/bin/env bash` shebang and `set -euo pipefail`
   - Set `SCRIPTS_DIR="$HOME/.claude/get-shit-done/scripts"`
   - Check if `$1` is `--legacy`:
     - If yes: set `TARGET="$SCRIPTS_DIR/autopilot-legacy.sh"`, shift the argument
     - If no: set `TARGET="$SCRIPTS_DIR/autopilot.mjs"`
   - Validate `TARGET` exists: if not, print error to stderr with the path and exit 1
   - For the legacy path: `exec bash "$TARGET" "$@"`
   - For the default path: `exec node "$TARGET" "$@"`
3. Verify the file is executable: `test -x bin/gsd-autopilot && echo "OK"`
4. Verify syntax: `bash -n bin/gsd-autopilot && echo "OK"`
</actions>
</task>

<task id="2">
<title>Update autopilot.mjs resume and escalation instructions</title>
<actions>

   - Line 135: `To resume: autopilot.sh --from-phase` -> `To resume: gsd-autopilot --from-phase`
   - Line 147: `To resume: autopilot.sh --from-phase` -> `To resume: gsd-autopilot --from-phase`

   - Line 439: `autopilot.sh --from-phase` -> `gsd-autopilot --from-phase`
   - Line 693: `autopilot.sh --from-phase` -> `gsd-autopilot --from-phase`
   - Line 783: `autopilot.sh --project-dir` -> `gsd-autopilot --project-dir`

</actions>
</task>
