# Plan 11-01: Gap Closure Loop in Autopilot

## Objective

Replace autopilot's current `exit 10` (gaps found) code paths with a gap closure loop that invokes `plan-milestone-gaps` to generate fix phases, executes them through the existing phase loop, re-audits, and repeats until the audit passes or a configurable iteration limit is exhausted — at which point it escalates to the human operator.

## must_haves

- [ ] When `run_milestone_audit` returns 10 (gaps found), autopilot invokes `/gsd:plan-milestone-gaps` via `run_step_with_retry` to generate fix phases
- [ ] The plan-milestone-gaps invocation includes an auto-approve directive so interactive confirmation is bypassed
- [ ] After gap planning completes, autopilot re-enters the existing phase loop by calling `find_first_incomplete_phase` to locate new fix phases and executing them through the normal discuss/plan/execute/verify/complete lifecycle
- [ ] After all fix phases complete (no more incomplete phases), autopilot automatically re-runs `run_milestone_audit`
- [ ] The audit-fix cycle repeats in a loop until `run_milestone_audit` returns 0 (passed) — no manual re-triggering needed
- [ ] An iteration counter tracks how many audit-fix cycles have been attempted, starting at 0 and incrementing before each gap planning invocation
- [ ] When the iteration counter reaches `max_audit_fix_iterations` (read from config), autopilot prints an escalation report and exits with code 1 instead of starting another fix cycle
- [ ] `max_audit_fix_iterations` is read via `get_config "autopilot.max_audit_fix_iterations" "3"` (defaults to 3)
- [ ] Both audit trigger points (startup all-complete path at line ~944 and main loop complete case at line ~1018) feed into the same gap closure loop
- [ ] Circuit breaker (`NO_PROGRESS_COUNT`) and iteration log (`ITERATION_LOG`) are reset before each gap closure iteration

## Context

**Two trigger points** currently exit with code 10 when gaps are found:

1. **Startup all-complete path** (lines 939-955): When `find_first_incomplete_phase` returns empty at startup, audit runs and `exit 10` on gaps.
2. **Main loop complete case** (lines 1013-1031): When `next_incomplete_phase` returns empty inside the main loop, audit runs and `exit 10` on gaps.

Both must call the same gap closure loop function instead of exiting.

**Existing helpers to reuse:**
- `run_milestone_audit` (line 200): Invokes audit, parses result, returns 0/10/1. Reuse directly for re-audit cycles.
- `run_step_with_retry` (line 467): Runs Claude steps with debug-retry. Use for gap planning invocation.
- `find_first_incomplete_phase` (line 689): Finds next phase to work on. Use after gap planning creates new fix phases.
- `get_config` (line 97): Reads config.json with defaults. Use for `max_audit_fix_iterations`.
- `print_halt_report` (line 147): Formatted halt output. Extend pattern for escalation report.
- `print_banner` (line 84): Banner formatting.

**Exit code contract:**
- Exit 0: audit passed (or tech debt accepted) — ready for Phase 12
- Exit 1: error or escalation (max iterations exhausted)
- Exit 10: NO LONGER propagated — consumed internally by the gap closure loop

**Config namespace:** `autopilot.max_audit_fix_iterations` alongside existing `autopilot.circuit_breaker_threshold`.

**plan-milestone-gaps interactive gate:** Step 5 asks "Create these {X} phases? (yes / adjust / defer all optional)". The prompt must include `--auto` or equivalent directive to auto-approve.

<task id="1" title="Add gap closure loop function and config reading">

### Description

Create a `run_gap_closure_loop` function in `autopilot.sh` that wraps the audit-fix cycle.

**Insert location:** After `run_milestone_audit` (around line 266), before the "Step Execution" section. This keeps audit-related functions together.

**Implementation:**

```bash
# ─── Gap Closure Loop ────────────────────────────────────────────────────────

run_gap_closure_loop() {
  local max_iterations
  max_iterations=$(get_config "autopilot.max_audit_fix_iterations" "3")
  local iteration=0

  while true; do
    # Check iteration limit BEFORE planning
    if [[ $iteration -ge $max_iterations ]]; then
      print_escalation_report "$iteration" "$max_iterations"
      exit 1
    fi

    iteration=$((iteration + 1))

    print_banner "GAP CLOSURE: Iteration $iteration/$max_iterations"

    # Reset circuit breaker and iteration log for fresh cycle
    NO_PROGRESS_COUNT=0
    ITERATION_LOG=()

    # LOOP-01: Invoke plan-milestone-gaps to create fix phases
    local gap_plan_prompt="/gsd:plan-milestone-gaps --auto"
    local gap_plan_exit=0
    run_step_with_retry "$gap_plan_prompt" "gap-planning" || gap_plan_exit=$?

    if [[ $gap_plan_exit -ne 0 ]]; then
      echo "ERROR: Gap planning failed (exit $gap_plan_exit)" >&2
      print_escalation_report "$iteration" "$max_iterations"
      exit 1
    fi

    # LOOP-02: Execute fix phases through existing phase loop
    while true; do
      local fix_phase
      fix_phase=$(find_first_incomplete_phase)

      if [[ -z "$fix_phase" ]]; then
        # All fix phases complete — break to re-audit
        break
      fi

      CURRENT_PHASE="$fix_phase"

      # Reset progress tracking for each fix phase
      NO_PROGRESS_COUNT=0
      ITERATION_LOG=()

      # Run the fix phase through the normal lifecycle
      while true; do
        local status_json step
        status_json=$(get_phase_status "$CURRENT_PHASE")
        step=$(echo "$status_json" | jq -r '.step')

        case "$step" in
          discuss)
            run_step "/gsd:discuss-phase $CURRENT_PHASE --auto" "discuss"
            ;;
          plan)
            run_step "/gsd:plan-phase $CURRENT_PHASE --auto" "plan"
            ;;
          execute)
            run_step_with_retry "/gsd:execute-phase $CURRENT_PHASE" "execute"
            local exec_exit=$?
            if [[ $exec_exit -ne 0 ]]; then
              print_halt_report "Fix phase execution failed after debug retries" "execute" "$exec_exit"
              exit 1
            fi
            ;;
          verify)
            run_verify_with_debug_retry "$CURRENT_PHASE"
            local verify_exit=$?
            if [[ $verify_exit -ne 0 ]]; then
              print_halt_report "Fix phase verification failed after debug retries" "verify" "$verify_exit"
              exit 1
            fi
            if [[ "$DRY_RUN" == true ]]; then
              echo "[DRY RUN] Auto-approving verification gate"
            else
              run_verification_gate "$CURRENT_PHASE"
            fi
            gsd_tools phase complete "$CURRENT_PHASE"
            ;;
          complete)
            print_banner "Fix Phase $CURRENT_PHASE COMPLETE ✓"
            break
            ;;
          *)
            echo "ERROR: Unknown step '$step' for fix phase $CURRENT_PHASE" >&2
            exit 1
            ;;
        esac
      done
    done

    # LOOP-03: Re-run milestone audit after fix phases complete
    print_banner "RE-AUDIT: After iteration $iteration"

    local audit_result=0
    run_milestone_audit || audit_result=$?

    if [[ $audit_result -eq 0 ]]; then
      # Audit passed — gap closure successful
      print_banner "GAP CLOSURE COMPLETE ✓"
      echo "Audit passed after $iteration iteration(s)."
      return 0
    elif [[ $audit_result -eq 10 ]]; then
      # LOOP-04: Gaps still found — loop continues
      echo "Gaps remain after iteration $iteration. Continuing..."
      continue
    else
      # Audit error
      echo "ERROR: Milestone audit encountered an error during re-audit" >&2
      exit 1
    fi
  done
}
```

### Escalation Report Function

Add `print_escalation_report` right after `print_halt_report` (around line 185):

```bash
print_escalation_report() {
  local iterations="$1"
  local max_iterations="$2"

  local audit_file
  audit_file=$(ls -t "$PROJECT_DIR/.planning/v"*"-MILESTONE-AUDIT.md" 2>/dev/null | head -1)

  echo ""
  echo "╔══════════════════════════════════════════════════════════════╗"
  echo "║  AUTOPILOT ESCALATION                                       ║"
  echo "╚══════════════════════════════════════════════════════════════╝"
  echo ""
  echo "Gap closure exhausted: $iterations/$max_iterations iterations completed"
  echo "Audit gaps persist after $iterations fix cycles."
  echo ""
  if [[ -n "$audit_file" ]]; then
    echo "Latest audit: $audit_file"
    echo ""
  fi
  echo "The autopilot could not converge to a passing audit."
  echo "Human review is required to diagnose remaining gaps."
  echo ""
  echo "To resume after manual fixes:"
  echo "  autopilot.sh --project-dir $PROJECT_DIR"
  echo ""
}
```
