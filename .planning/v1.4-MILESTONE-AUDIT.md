### Integration Gap

**CMD-01 portability defect:** `commands/gsd/linear.md` uses hardcoded `@/Users/seanspade/.claude/get-shit-done/workflows/linear.md` in both `<execution_context>` and `<process>` blocks. All other command specs use `@~/.claude/get-shit-done/workflows/...` pattern. The installer's `copyWithPathReplacement` replaces `~/.claude/` tokens but cannot process absolute paths. This makes the command non-portable for other users and non-Claude runtimes.
